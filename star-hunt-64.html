Star Hunt 64 SM64

A Super Mario 64 decompilation and modification project.

Setup in Progress

About

Star Hunt 64 is a decompilation project based on n64decomp/sm64, focusing on understanding the original Super Mario 64 codebase and exploring potential modifications.

This project builds the US version of Super Mario 64 using the decompiled source code.

Requirements

Legal ROM Required

You must own a legal copy of Super Mario 64 (US version). The ROM file is not included and must be provided by the user.

  • macOS with Homebrew or Docker
  • MIPS cross-compilation toolchain (mips64-elf-binutils)
  • ~2GB free disk space

Setup (macOS)

Step 1: Install Dependencies

brew update
brew install coreutils make pkg-config tehzz/n64-dev/mips64-elf-binutils

Step 2: Add Baserom

Place your legal ROM copy at the project root:

cp /path/to/sm64.us.z64 baserom.us.z64

Step 3: Build

cd projects/star-hunt-64
gmake VERSION=us -j4
